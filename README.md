
# EasyCode

> 每天十行烂代码,项目维护到终老

本项目用于学习优化代码,来跟我一起修练吧

======================================================================

- 代码写的好，bug少,看起来就像个闲人。
- 注释多，代码逻辑清晰，任何人接手都很方便，看起来谁都可以取代。
- 代码写的烂，每天风风火火改bug，各种救火，解决各种线上重大问题，顺理成章成了公司
亮眼的人才。
- 代码乱的只有自己看得懂，公司不可替代的人才。
